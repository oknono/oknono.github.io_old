I going to try to give some more frequent updates about what I am working on at the Recurse Center. This week marks the beginning of the 5th week, and boy time sure does fly.

Week 1 and 2 : The first two weeks were mostly spend on settling in and getting used to the new place, the new people and the new time zone. The older I can the less I can handle things like jet lag, it’s pathetic but true. There were workshops on cryptography ( by Frank Wang), working with git, an introduction to Haskell, algorithms and some interaction design. I started this cryptography course on Coursera, and took my first look at JavaScript

Week 3: Continued with the cryptography course. I paired on my first project writing a simple coin toss in JavaScript. I did a JavaScript tutorial on audacity and started with the online course/book eloquent JavaScript. Besides JavaScript I spend some time working on small python exercises in vim to keep both my python skills up and to get some more exposure and exercise in using vim. Other than that I learned some new bash skills and make my first pull request in Github.

Week 4: Two workshops by Allison Parrish, one on Twitter-bots and another one on the Carnegie melon pronunciation library. If I make some progress with building some projects with JavaScript, I’d like to look at some natural language processing in Python. I did some more work with Eloquent JavaScript and started a tutorial on D3. I took a look at sketch for designing the interfaces for my projects. I finally got some practice and skills in reading and writing to and from files in Python.

Week 5: My plans for this week are as follows
- Work on a project using JavaScript (libraries) / probably an Online I ching
- Keep working on Eloquent JavaScript - try to get to the chapters with projects.
- Find a nice data set for a D3 project
- Continue with the algorithms course
- Quit the cryptography course for now
- Keep working on small exercises in Python using Vim
